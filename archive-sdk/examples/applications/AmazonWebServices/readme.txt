This application makes use of a free Amazon Web Service.   To get started in using the free service, you must sign up at Amazon for a developers token, which one must enter into the Application Server management front-end, "Application Configuration" section of the installed application.

The registration can be found here:

http://www.amazon.com/gp/browse.html/002-8510643-2349628?node=3435361

If invalid, search using a search engine on "Amazon web service" (not in quotes).  It should be near the top of the results.
