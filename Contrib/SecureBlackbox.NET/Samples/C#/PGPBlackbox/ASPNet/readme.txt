The sample illustrates the use of PGPBlackbox components in ASP.NET environment.

Please copy the sample to your Inetpub directory (%HOMEDRIVE%\Inetpub\wwwroot\ by default).
Put your keyring files to the 'keys' directory (public and secret keyring files MUST have
'pubring.pkr' and 'secring.skr' names respectively). Remember to enable write access 
to the 'files' directory.
