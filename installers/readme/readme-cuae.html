CISCO UNIFIED APPLICATION ENVIRONMENT SOFTWARE RESTORATION

This document outlines the sequence of steps required to re-image a Cisco Unified Application Environment (CUAE) Server.

BEFORE YOU BEGIN

Note: The Cisco Unified Application Environment Server ships from Cisco Pre-Installed with all the software required for operation. This procedure should *only* be executed if you need to re-Image the server. Executing this procedure will erase all existing data and configuration information on your server.

This procedure requires:

  • Cisco MCS Server 7845-H
  • Cisco MCS OS DVD version 2003.1.0.5002-H
  • Cisco Unified Application Environment restore DVD version 2.3

MCS OS DVD INSTALLATION

The MCS OS DVD will install the needed Windows 2003 Server OS image necessary for the CUAE. The MCS OS DVD is completely automated:

  1. Insert the DVD and reboot the unit
  2. Acknowledge all prompts until the installation is complete

CUAE DVD INSTALLATION

After the MCS OS DVD has successfully installed, insert the CUAE DVD and begin the CUAE installation process. This process will install all of the necessary software for the Cisco Unified Application Server and Cisco Unified Media Engine.

CORE ADDENDUM INSTALLER CISCO UNIFIED MCS OS CORE ADDENDUM.EXE

Run the file to install the Core Addendum pieces. This will install the following 3rd party applications:

  • JAVA in C:\Program Files\Java
  • PHP in C:\php
  • OpenSSL in C:\openssl
  • OpenSpeech Recognizer Client
  • Microsoft WSE 2.0 SP2 Runtime
  • Installs and configures the following services:
    • Apache Webserver
    • Intel HMP service as ‘Dialogic’
    • NeoSpeech as ‘VT Server’
    • Installs and disables Neospeech ‘VTAdmin Server’ service
    • Disables the ‘IIS Admin’ service
    • MySQL service and ODBC drivers

Note: At the end of this installer the user must reboot the system.

PLATFORM INSTALLER - CISCO UNIFIED APPLICATION PLATFORM SERVICE.MSI

This installer creates the CiscoUAE user for running the services. It will also install all of the common support services required for the Unified Application Server and the Unified Media Engine.

  • The installer will prompt for a password for the newly created CiscoUAE user. The password needs to be entered twice.
  • The following services are installed and configured for DB access. These services are not started automatically as part of the installation.
    • CUAE Alarm server
    • CUAE Log server
    • CUAE Watchdog service
    • CUAE SFTP service

APPLICATION SERVER INSTALLER - CISCO UNIFIED APPLICATION SERVER.MSI

Note: This installer will not run if there is no CiscoUAE user (i.e. if the Platform Installer has never run successfully)

  • The installer will ask for the CiscoUAE user password. It must be entered twice for verification.
  • The application server installer installs and configures:
    • CUAE - Application server service
    • CUAE - JTapi Provider services
    • CUAE - H323 Provider service
    • CUAE - SIP Provider service
    • CUAE - SCCP Provider service

CISCO UNIFIED MEDIA ENGINE INSTALLER - CISCO UNIFIED MEDIA ENGINE.MSI

Note: This installer will not run if there is no CiscoUAE user (i.e. if the Platform installer has never run successfully)

  • The installer will ask for the CiscoUAE user password. It must be entered twice for verification.
  • The Media Server installer installs and configures:
    • CUAE – Media Engine service

COMPLETE

Server Re-Image is now complete.
